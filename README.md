# 停车场管理系统 java569

## 项目概述

停车场管理系统是一款基于Spring Boot和Vue.js的前后端分离系统。该系统整合了MyBatis Plus和MySQL技术栈，实现了车位管理、预定、缴费等一系列功能。

## 技术栈

- 前端：Vue.js 2.x
- 后端：Spring Boot
- 数据库访问：MyBatis Plus
- 数据库：MySQL
- 前后端通信：RESTful API

## 功能模块

### 用户功能

- 个人中心
- 预定停车位
- 缴费信息

### 管理员功能

- 用户信息管理
- 车位信息管理
- 车位费用管理
- 停泊车辆查询
- 车辆进出管理
- 登录日志查询

## 系统角色

- 管理员
- 用户

## 运行环境

- 前端：现代浏览器
- 后端：Java 8+
- 数据库：MySQL 5.7+

## 部署步骤

1. 安装Java环境
2. 安装MySQL数据库，并导入SQL文件
3. 部署后端服务
4. 部署前端服务
5. 访问前端地址，开始使用
